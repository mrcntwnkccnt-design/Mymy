🎯 ОСНОВНЫЕ КОМПОНЕНТЫ
═══════════════════════════════════════════════════════════════════════════════

ACTIVITIES:
1. SplashActivity
   - Проверка авторизации при запуске
   - Автоматический вход или переход на экран входа

2. AuthActivity
   - Регистрация нового аккаунта
   - Вход с email/пароль
   - Переключение между режимами
   - Валидация данных

3. MainActivity
   - Главное окно приложения
   - Bottom Navigation (3 вкладки)
   - FragmentContainer для переключения

FRAGMENTS:
1. ChatsFragment
   - Список всех чатов
   - Последнее сообщение и время
   - Счетчик непрочитанных
   - Кнопка logout

2. SearchFragment
   - Поиск пользователей по ID
   - Отображение профиля
   - Кнопка "Start Chat"

3. ChatFragment
   - Окно переписки с пользователем
   - Отправка сообщений
   - Получение сообщений в реальном времени
   - Список сообщений

ADAPTERS:
1. ChatsAdapter
   - RecyclerView для списка чатов
   - Элементы с last message
   - Unread badges

2. MessagesAdapter
   - RecyclerView для сообщений
   - Различие между отправленными/полученными
   - Временные метки

MODELS:
1. User
   - userId, username, email
   - createdAt, lastSeen

2. Message
   - messageId, senderId, recipientId
   - content, timestamp, isRead

3. Chat
   - chatId, userId, otherUserId
   - lastMessage, lastMessageTime, unreadCount

REPOSITORIES:
1. AuthRepository
   - register(email, password, username)
   - login(email, password)
   - getCurrentUser()
   - logout()
   - isUserLoggedIn()

2. ChatRepository
   - searchUserById(userId)
   - sendMessage(recipientId, content)
   - getChats()
   - getMessages(otherUserId)


💾 FIREBASE СТРУКТУРА
═══════════════════════════════════════════════════════════════════════════════

COLLECTIONS:

/users/{userId}
  ├─ userId (string)         - Firebase UID
  ├─ username (string)       - Имя пользователя
  ├─ email (string)          - Email
  ├─ createdAt (number)      - Timestamp создания
  └─ lastSeen (number)       - Timestamp последнего входа

/chats/{chatId}
  ├─ chatId (string)         - Уникальный ID чата
  ├─ userId (string)         - ID основного пользователя
  ├─ otherUserId (string)    - ID другого пользователя
  ├─ otherUserName (string)  - Имя другого пользователя
  ├─ lastMessage (string)    - Последнее сообщение
  ├─ lastMessageTime (number)- Время последнего сообщения
  └─ unreadCount (number)    - Количество непрочитанных

/chats/{chatId}/messages/{messageId}
  ├─ messageId (string)      - Уникальный ID сообщения
  ├─ senderId (string)       - ID отправителя
  ├─ recipientId (string)    - ID получателя
  ├─ content (string)        - Текст сообщения
  ├─ timestamp (number)      - Время отправки
  └─ isRead (boolean)        - Прочитано ли


🚀 КАК НАЧАТЬ
═══════════════════════════════════════════════════════════════════════════════

БЫСТРЫЙ СТАРТ (5 минут):

1. Firebase Setup
   → Откройте https://console.firebase.google.com
   → Создайте проект "android-messenger-app"
   → Включите Email/Password Authentication
   → Создайте Firestore Database
   → Скачайте google-services.json
   → Поместите в AndroidMessenger/app/

2. Сборка
   → cd /workspaces/Mymy/AndroidMessenger
   → ./gradlew clean assembleDebug

3. Установка
   → adb install -r app/build/outputs/apk/debug/app-debug.apk

4. Запуск
   → adb shell am start -n com.example.androidmessenger/.ui.activity.SplashActivity

5. Использование
   → Создайте аккаунт
   → Запомните User ID
   → Найдите другого пользователя
   → Начните переписку!

✅ Проверьте Firestore Rules
